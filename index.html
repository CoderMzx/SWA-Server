Swa-server

简单高效Socket框架-可用于中小型游戏、应用开发（拿来即可使用）

Download this project as a .zip file Download this project as a tar.gz file

欢迎来到 Swa-server 开源项目

适用于中小型游戏，养成、RPG、棋牌等；应用软件，如：聊天室等

已经封套好底层socket管理，sql请求处理、数据加密解密

拿来即可开工写业务

Swa-server是面向小型快速开发的框架，所以采用单进程模式，以后也是单进程，减少开发时间与人力（其实很多服务器用单进程就够了）

----------------------------请下文详细介绍-----------------------------------------------------------------

支持IO异步（基于boost库IO）

支持sql异步执行（有回调函数）

玩家管理器

数据库管理器
